2011-2025  Event Editor 
To the immediate right of the Hand Editor, the Event Editor displays a list of every 
action
 
that
 
occurs
 
during
 
the
 
currently
 
selected
 
hand.
 
●
 
To
 
add
 
a
 
new
 
event,
 
click
 
the
 
‘New’
 
button.
 
The
 
new
 
event
 
will
 
be
 
inserted
 
immediately
 
after
 
the
 
currently
 
selected
 
event,
 
if
 
any.
  
●
 
To
 
delete
 
an
 
event,
 
click
 
the
 
‘Del’
 
button.
  
●
 
To
 
change
 
an
 
existing
 
event,
 
click
 
the
 
‘Edit’
 
button.
  
●
 
The
 
‘Linger’
 
value
 
determines
 
how
 
long
 
the
 
graphics
 
remain
 
on-screen
 
after
 
the
 
last
 
event
 
has
 
occurred
 
–
 
for
 
example
 
after
 
a
 
final
 
CALL
 
event.
 
The
 
default
 
period
 
is
 
3,000
 
milliseconds.
  
/g0
 
To
 
make
 
the
 
graphics
 
disappear
 
at
 
a
 
specific
 
moment
 
in
 
the
 
video,
 
navigate
 
to
 
the
 
correct
 
moment
 
with
 
the
 
Player
 
controls,
 
and
 
then
 
click
 
on
 
the
 
‘Linger’
 
box
 
while
 
holding
 
down
 
the
 
ALT
 
key.
  
●
 
To
 
change
 
the
 
time
 
an
 
event
 
occurs,
 
use
 
the
 
Player
 
controls
 
to
 
move
 
the
 
video
 
to
 
the
 
correct
 
moment
 
in
 
the
 
video,
 
and
 
then
 
click
 
on
 
the
 
event
 
to
 
move
 
while
 
holding
 
down
 
the
 
ALT
 
key.
  
/g0
 
No
 
checking
 
is
 
performed
 
on
 
the
 
validity
 
of
 
events.
