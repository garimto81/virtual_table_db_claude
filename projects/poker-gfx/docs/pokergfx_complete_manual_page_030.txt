Supported recording modes 
The recording mode can be selected from the ‘Outputs’ settings tab: 
Game data only (no video). Typically used for rendering hands in post production where game video has been recorded outside PokerGFX. Game data with video and GFX. This is the same as the live stream. Game data with video but no GFX. This is commonly used when the game is streamed live with graphics, but ‘clean’ video is recorded with no graphics. Graphics are added in post production using the recorded hand data.
